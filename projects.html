Melee Calculator

This is a personal ongoing JavaFX project that had a bulk of the work done in the summer of 2017. This program is used to calculate various useful things for competitive Super Smash Bros. Melee (video game) players, such as knockback, hitstun, hitlag, stale move negation, and frame advantage on block, hit, and AT.
